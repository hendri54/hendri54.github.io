A Database of Mincerian Earnings Regressions

Purpose

Provide a set of Mincerian earnings regressions for a broad range of countries and years in easily accessible format. The data will be continuously updated.

Documents

Excel spreadsheet containing Mincerian regression coefficients and information on sample characteristics.

Documentation for individual sources. Also a list of references.

Notes to users

If you use this data, please cite  the source as

Hendricks, Lutz (2004). "A Database of Mincerian Earnings Regressions." www.lhendricks.org/Mincer.htm

Please contribute links to additional studies and sources.

Please notify me of any errors or other corrections that need to be made.

Please let me know of papers using this data. It helps me write grants to support this effort.

Credits
