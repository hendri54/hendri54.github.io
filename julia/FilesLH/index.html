FilesLH.deploy_docs — Function
deploy_docs(pkgDir; trialRun, tgBaseDir)

Deploy package docs to my website.

This is to be included in each package's make.jl. The benefit over hosting on github is to work around issues with private repos and private registries that would arise when TravisCI is used.

Example

# This gives the package directory, if called from `docs/make.jl`.
pkgDir = rstrip(normpath(@__DIR__, ".."), '/');
@assert endswith(pkgDir, "FilesLH")
deploy_docs(pkgDir);

Arguments

  • pkgDir: points to package directory. If pkgDir = /a/b/Foo the docs end up in julia/Foo on my website.

FilesLH.rsync_command — Function
rsync_command(srcDirIn, tgDirIn; trialRun, doDelete, doCompress, verbose, inclSubDirs, progress, excludeV)

Rsync to or from remote machine. Only for directories.

Fails if local or remote paths contain spaces.

Arguments

  • srcDirIn, tgDirIn :: String Source and target directories. Remote directory must include part required for ssh (e.g. lhendri@longleaf.unc.edu:)
  • doDelete :: Bool Should orphan files be deleted on target? False by default.
  • trialRun :: Bool
  • progress :: Bool Show progress indicator

FilesLH.make_remote_dir — Function
make_remote_dir(remoteDir, sshStr; trialRun, createParents)

Make a directory on the remote, optionally creating all parents as well. Creates a command of the form ssh lhendri@longleaf.unc.edu 'mkdir -p ~/home/abc'. Shows on-screen error if the directory already exists.

Note that other shell commands can be run with the same syntax.
