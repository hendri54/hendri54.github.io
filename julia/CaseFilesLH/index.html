CaseFilesLH

Handles names and directories for named "Cases" that are typically used to run computations with counterfactuals.

A CaseName consists of a base name, modifiers, and experiment modifiers.

From it, subdirectories are constructed using relative_dir.

CaseNameDirectory
:default"default/base"
:default, [:mod1]"default/mod1"
:default, [:mod1, :mod2], [:xp1, :xp2]"default/mod1mod2/experiments/xp1xp2"
CaseFilesLH.CaseName — Type

A CaseName contains the base name, modifiers for the base name, and modifiers for counterfactuals (or experiments).
